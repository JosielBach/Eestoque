API de Gerenciamento de Estoque<br/>
API desenvolvida em .NET (C#) como projeto de estudo, com foco na aplicação de boas práticas de arquitetura, organização de código e construção de APIs escaláveis.

Destaques<br/>
Arquitetura baseada em Clean Architecture<br/>
Padrões Repository e Unit of Work<br/>
Uso de Entity Framework Core para persistência de dados<br/>
DTOs + AutoMapper para desacoplamento entre camadas<br/>
Tratamento de exceções com middleware global<br/>
Programação assíncrona com async/await

Objetivo<br/>
Consolidar conhecimentos no desenvolvimento de APIs modernas com .NET, aplicando princípios como SOLID, separação de responsabilidades e boas práticas de mercado.
